# WRecode converter

Adds ffmpeg conversion with GUI to the context menu for easy and fast media file conversion.

## How to use:
Make sure you have installed [python](https://www.python.org/downloads/windows) or from Microsoft Store https://apps.microsoft.com/detail/9pnrbtzxmb4z?hl=en-US&gl=US

> [!IMPORTANT]
> If you move WRecode folder, reinstall app with **uninstall.py**, and **install.py** again from new folder
> 
> Make sure your user name not contain space, if it is, you may start WRecode convert.bat file manually.
> Copy file from C:\Users\user\AppData\Roaming\Microsoft\Windows\SendTo\WRecode convert.bat.
> Drag&drop media on it and it works


> [!WARNING]
> Do not rename any WRecode files, script will be broken
